Business Requirements Document (BRD)

Project Name

Study Group Finder

Objective

To help students create, find, and join study groups based on subjects or courses.

Functional Requirements

- User registration and login.
- Create study groups.
- Search groups by subject or course.
- Join and leave study groups.
- Set a maximum member limit.
- Add study schedules.
- Add meeting locations or online links.

Non-Functional Requirements

- Secure user authentication.
- Simple and responsive user interface.
- Fast application performance.
- Support mobile and desktop browsers.

Must-Have Features

- Authentication.
- Group creation.
- Group search.
- Join and leave groups.
- Member limit management.
- Study scheduling.

Nice-to-Have Features

- Group chat.
- Notifications.
- Dark mode.
- AI-based group recommendations.
